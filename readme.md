# **Utilidades Android**

Este proyecto es un ejemplo de la aplicación de algunas de las herramientas que android ofrece para el desarrollo de aplicaciones.

1. **Formularios y validaciones**
2. **Manejo de eventos**
  * Click
  * Cambios de foco
  * Cambios de contenido en input.
3. **Manejo de sensores**
4. Integración con el API de Google Maps.
5. Localización.
6. Información de la red telefónica.
